# 建立封裝加速器頁面


使用 [**建立套件加速器**] 頁面來指定要儲存套件加速器的位置。

此頁面包含下列元素：

<a href="" id="browse"></a>**瀏覽**  
按一下 **[流覽]** 以指定要儲存套件加速器的位置。
