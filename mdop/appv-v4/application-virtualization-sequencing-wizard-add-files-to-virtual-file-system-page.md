# 應用程式虛擬化順序嚮導 [新增檔案至虛擬檔案系統] 頁面


使用 [應用程式虛擬化順序] 嚮導中的 [**新增檔案至虛擬檔案系統**] 頁面，選取要新增至虛擬檔案系統的檔案。 此頁面包含下表所述的元素。

</tr>
<tr class="even">

<td align="left"><p>按一下以從清單中移除選取的檔案，不會將其新增至虛擬檔案系統。</p></td>
