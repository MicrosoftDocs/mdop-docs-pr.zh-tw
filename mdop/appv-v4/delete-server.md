# DELETE SERVER


移除發佈伺服器。

**記事** 這個命令不會移除伺服器發佈至用戶端的任何應用程式或套件。 針對每個應用程式，使用 SFTMIME **CLEAR APP**命令，後面接著 [**刪除套件**] 命令，將這些應用程式和套件從用戶端中完全移除。

 

`SFTMIME DELETE SERVER:server-name [/LOG log-pathname | /CONSOLE | /GUI]`
