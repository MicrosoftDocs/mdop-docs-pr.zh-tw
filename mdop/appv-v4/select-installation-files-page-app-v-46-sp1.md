# 選取 [安裝檔案] 頁面


使用 [**選取安裝**檔案] 頁面來選取所選套件加速器所需的安裝檔。 您指定的安裝檔應該位於執行 App-v 排序器的電腦上。 所需的安裝檔也會顯示在此頁面上。

此頁面包含下列元素：

<a href="" id="browse"></a>**瀏覽**  
按一下 **[流覽]** ，指定內含所選套件加速器所需的安裝檔案的目錄，以建立虛擬應用程式套件。 目錄應該位於執行排序器的電腦上。 您可以指定下列媒體和格式類型的安裝檔：

-   壓縮檔

-   Windows Installer 檔案

-   .cab 檔案

-   本機檔案和目錄
