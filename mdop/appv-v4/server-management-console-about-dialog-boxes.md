# 伺服器管理主控台：關於對話方塊


[**關於**] 對話方塊會顯示特定版本的 Microsoft 管理主控台或應用程式虛擬化伺服器管理主控台的特定版本資訊。

若要顯示這些對話方塊，請**按一下 [** 說明]，然後選取 [**關於 Microsoft 管理主控台**] 或 [關於**應用程式虛擬伺服器管理主控台**]。
