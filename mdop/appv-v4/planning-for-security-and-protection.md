# 規劃安全性與保護


做為基礎結構管理員，您必須提供安全、高效且可支援的環境。 Microsoft Application Virtualization （App-v）系統的安全性依賴正確設定軟體及其運作的環境。 此區段的規劃與部署指南提供設定各種 App-v 元件的指導方針，以增強您環境的安全性。 在決定要部署系統之前，請先仔細考慮 App V 環境的曝露與攻擊面，而不要套用本節中所述的安全性建議。
