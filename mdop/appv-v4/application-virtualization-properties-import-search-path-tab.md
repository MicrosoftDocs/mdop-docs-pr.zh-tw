# Application Virtualization 內容：匯入搜尋路徑索引標籤


使用 [**應用程式虛擬化屬性**] 對話方塊的 [匯**入搜尋路徑] 索引**標籤來查看及管理匯入 SFT 檔案的搜尋路徑。

此索引標籤包含下列元素。

<a href="" id="path-window"></a>**路徑視窗**  
這個視窗會顯示 SFT 路徑。

<a href="" id="add-and-remove"></a>**新增**及**移除**  
使用這些按鈕來新增或移除 SFT 搜尋路徑。

<a href="" id="move-up-and-move-down"></a>[**上移** **] 和 [下移]**  
使用這些按鈕來整理 SFT 搜尋路徑。
