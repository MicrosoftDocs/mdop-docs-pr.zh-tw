# [套件名稱] 頁面


使用 [**套件名稱**] 頁面來指定要與新的虛擬應用程式套件建立關聯的名稱。

此頁面包含下列元素：

<a href="" id="virtual-application-package-name"></a>**虛擬應用程式套件名稱**  
指定描述新虛擬應用程式套件的名稱。 此名稱也會識別 App-v 管理主控台中的虛擬應用程式套件。
