# 執行每個程式頁面


使用 [**執行每個程式**] 頁面來執行每個程式，以在安裝後完成任何設定工作。 例如，此步驟可協助設定任何相關聯的應用程式授權協定。 在 [**應用程式] 窗格**中選取應用程式，然後選取下列其中一個選項。

此頁面包含下列元素：

<a href="" id="run-selected"></a>**執行選取的**  
只執行選取的應用程式。

<a href="" id="run-all"></a>**全部執行**  
執行套件中包含的所有應用程式。
