# 如何建立伺服器群組


為了協助您更有效率地管理您的應用程式虛擬化管理伺服器，您可以使用應用程式虛擬化伺服器管理主控台將這些伺服器組織在伺服器群組中。

**建立伺服器群組**

1.  按一下左窗格中的 [**伺服器群組**] 節點，然後選擇 [**新增伺服器群組**]。

2.  在 [**新增伺服器群組] 嚮導**中，在 [**伺服器群組名稱**] 欄位中輸入名稱，然後從提供者原則的下拉式清單中選取預設的提供者原則。

3.  請確定已選取 [**啟用**] 核取方塊，以啟用伺服器群組。

4.  按一下 **\[完成\]**。
