# 伺服器管理主控台中的 [應用程式結果] 窗格欄


應用程式虛擬化管理主控台中的 [**應用程式結果**] 窗格會顯示各種資料行。

您可以使用標準的 Microsoft Windows [**新增/移除欄**] 對話方塊來選取 [**結果**] 窗格中顯示的欄。

若要查看 [**新增/移除欄**] 對話方塊，請在 [**結果**] 窗格中按一下滑鼠右鍵，然後從快顯功能表中選取 [查看]，然後選取 [ ** &gt; 新增/移除欄**]。
