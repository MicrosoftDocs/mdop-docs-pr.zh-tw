# 不相容的安裝程式對話方塊 (App-V 4.6 SP1)


無法正確安裝應用程式。

使用下列程式驗證您嘗試順序的應用程式是否與執行排序器的電腦相容。

1.  關閉 App-v 排序器。

2.  在執行 Sequencer 的電腦上安裝應用程式，而不監視安裝。

3.  確認應用程式是相容的且可以成功安裝。
