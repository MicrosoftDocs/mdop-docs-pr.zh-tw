# 選取封裝 (深入瞭解) 頁面


使用 [**選取套件**] 頁面，選取您要用來建立套件加速器的來源虛擬應用程式套件。 將您在此頁面上指定的套件複製到執行排序器的電腦上。

此頁面包含下列元素：

<a href="" id="browse"></a>**瀏覽**  
按一下 **[流覽]** 以指定您將用來建立套件加速器的現有虛擬應用程式套件。 在執行排序器的電腦上，將您指定的套件儲存在本機。
