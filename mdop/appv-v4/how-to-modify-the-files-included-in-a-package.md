# 如何修改封裝中包含的檔案


當您將應用程式排序之後但在儲存前，您可以手動修改已排序的應用程式套件，以指定它將保留其設定的方式。 使用 [Sequencer 屬性] 區段，指定已排序之應用程式套件中的檔案是否可供特定使用者或群組中的所有使用者使用。

**修改 Sequencer 屬性**

1.  按一下 [**檔案**] 索引標籤。

2.  根據您想要讓應用程式保留其設定的方式，選取 [**使用者資料**] 或 [**應用程式資料**]，然後選取 [覆**寫**]，讓用戶端覆寫快取的應用程式。

3.  按一下 **\[Apply\]** (套用)。
