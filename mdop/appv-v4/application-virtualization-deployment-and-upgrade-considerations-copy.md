# Application Virtualization 部署與升級考量


開始部署 Microsoft 應用程式虛擬化（App-v）之前，您可能必須審查您的環境需求，包括安裝各種應用程式虛擬化元件的硬體和軟體需求。 此外，如果您是從舊版升級，本節中的主題會提供有關如何升級您目前的排序器、伺服器和用戶端版本的資訊。
