# Application Virtualization Sequencer 概觀


Microsoft Application Virtualization （App-v） Sequencer 會監視及記錄應用程式的所有安裝與設定處理常式，並建立下列檔案： **.ico**、 **OSD**、 **SFT**及**SPRJ**。 這些檔案包含應用程式的所有必要資訊，讓應用程式可以在虛擬環境中執行。
