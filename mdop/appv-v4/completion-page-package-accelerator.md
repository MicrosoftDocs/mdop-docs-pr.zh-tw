# 完成頁面


使用 [**完成**] 頁面來查看套件完成報告。 在排序期間遇到的任何問題都會顯示在 [**成功的虛擬應用程式套件報告**] 窗格中。 此資訊也會在將套件儲存在名為 Report.xml 的檔案中的目錄中提供。

此頁面包含下列元素：

<a href="" id="close"></a>**關閉**  
關閉 [**建立新套件] 嚮導**。
