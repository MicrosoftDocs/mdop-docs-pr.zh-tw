# 如何在伺服器管理主控台中管理應用程式授權


應用程式虛擬化伺服器管理主控台是您用來管理應用程式虛擬化平臺的介面。 從它來看，您可以新增、移除、設定及控制應用程式授權群組。

**重要** 如果應用程式-V 用戶端應用程式來源根（ASR）設定已設定為使用管理伺服器以外的任何資料流程來源類型（例如，流式伺服器、IIS 伺服器或檔案伺服器），則管理伺服器無法強制執行其授權原則。
