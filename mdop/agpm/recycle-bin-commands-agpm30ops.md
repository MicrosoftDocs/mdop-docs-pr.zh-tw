# 資源回收筒命令


[**回收站**] 索引標籤：

-   顯示已從封存中刪除的群組原則物件（Gpo）清單。

-   提供快捷方式功能表，其中包含管理 Gpo 及顯示 Gpo 報告的命令。

-   顯示有許可權存取所選 GPO 的群組和使用者清單。

以滑鼠右鍵按一下此索引標籤上的 [**群組原則物件**] 清單，即可顯示快捷方式功能表，包括下列任何一項適用的選項：
