# 建立範本和設定預設範本


建立範本可讓您儲存特定版本的群組原則物件（GPO）的所有設定，以做為建立新 Gpo 的起點。 做為編輯者，您也可以指定哪些可用範本將是所有建立新 Gpo 之群群組原則系統管理員的預設範本。

**記事** 範本是一個無法編輯且靜態的 GPO 版本，以做為建立新的可編輯 Gpo 的起點。 重新命名或刪除範本不會影響從該範本建立的 Gpo。

-   [設定預設範本](set-a-default-template.md)
