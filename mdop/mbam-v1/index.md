# Microsoft BitLocker 管理和監視1管理員指南

Microsoft BitLocker 管理與監控（MBAM）提供簡化的管理介面，您可以用來管理 BitLocker 磁碟機加密。 有了 MBAM，您就可以選取適合您企業的 BitLocker 加密原則選項，然後使用這些策略來監視用戶端是否符合這些原則。 您也可以報告個別電腦與整個企業的加密狀態。 此外，您也可以在使用者忘記其 PIN 或密碼時，或當他們的 BIOS 或開機記錄變更時，存取復原金鑰資訊。
