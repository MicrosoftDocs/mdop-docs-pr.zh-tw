# MED-V 概觀


MED-V 使用 Microsoft 虛擬電腦為桌面虛擬化提供企業解決方案。 有了 MED-V，您可以輕鬆地在任何採用 Windows® 的桌面上建立、交付和管理公司虛擬電腦映像。

MED-V 是 Microsoft 桌面優化套件的一個有機元件，可供軟體保證客戶使用的動態解決方案，可協助減少應用程式部署成本，讓應用程式可以傳送成服務，並協助您更好地管理及控制企業桌面環境。
