# 如何安裝 MED-V 用戶端


在使用者的電腦上安裝 MED-V 套件之前，請確定圖像已上傳至網路，或已推至影像的暫存資料夾。

若要使用企業軟體發行中心安裝 MED-V 用戶端，請參閱如何使用您正在使用的系統來安裝應用程式的相關檔。
