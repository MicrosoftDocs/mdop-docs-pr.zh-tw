# 如何使用命令列安裝 MBAM 用戶端


您可以使用命令列來部署 Microsoft BitLockerAdministration 和 Monitoring （MBAM）2.0 用戶端。

## <a href="" id="command-line-for-deploying-the-mbam-2-0-client-"></a>部署 MBAM 2.0 用戶端的命令列


使用下列命令列來部署 MBAM 用戶端：

`MBAMClientSetup.exe /q`
