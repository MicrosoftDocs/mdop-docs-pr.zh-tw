# 建立 DaRT 10 復原映像


安裝 Microsoft Diagnostics 和恢復工具集（DaRT）10之後，您會建立一個 DaRT 10 復原影像。 [恢復] 影像會啟動 Windows RE，您可以從這裡啟動 DaRT 工具。 您可以產生國際標準組織（ISO）檔案和 Windows Imaging 格式（WIM）影像。 此外，您可以使用 PowerShell 來產生使用您在 DaRT 復原映像嚮導中所選取之設定的腳本。 您可以稍後使用此腳本，透過使用相同的設定來重建復原影像。 復原影像提供各種不同的恢復工具。 如需工具的描述，請參閱[DaRT 10 中的工具概覽](overview-of-the-tools-in-dart-10.md)。

將電腦引導至 DaRT 之後，您可以執行不同的 DaRT 工具來嘗試診斷並修復電腦。 本節將引導您逐步完成建立 DaRT 復原影像的程式，並讓您選取要納入影像中的工具和功能。

您可以使用下列兩種方法的其中一種來建立 DaRT 復原影像：

-   使用 DaRT 復原映射嚮導（在 Windows 環境中執行）。

-   使用您想要的值修改範例 PowerShell 腳本。 如需詳細資訊，請參閱[如何使用 PowerShell 腳本來建立恢復影像](how-to-use-a-powershell-script-to-create-the-recovery-image-dart-10.md)。

您可以將 ISO 寫入可錄製的 CD 或 DVD、將其儲存至 USB 快閃磁片磁碟機，或將它儲存為可從遠端分區或從復原分區引導至 DaRT 的格式。

建立 ISO 影像之後，您可以將它燒錄到空白的 CD 或 DVD （如果您的電腦有 CD 或 DVD 磁片磁碟機）。 如果您的電腦沒有用於這種用途的磁片磁碟機，您可以使用大多數用來燒錄 Cd 或 Dvd 的一般程式。

## 選取影像架構並指定路徑


在 [Windows 10 媒體] 頁面上，您可以選取是否要建立32或64位的 DaRT 恢復影像。 使用32位 Windows 來建立32位的 DaRT 復原影像，以及64位的 Windows 來建立 64-位 DaRT 恢復影像。 您可以使用單一電腦來為這兩種架構類型建立復原影像，但無法建立一個在32位與64位架構上都能運作的影像。 您也會指出 Windows 10 安裝媒體的路徑。 選擇與您要建立之其中一個恢復影像相符的架構。

**選取影像架構並指定路徑**

1.  在 [ **Windows 10 媒體**] 頁面上，選取下列其中一項：

    -   如果您要建立64位電腦的復原影像，請選取 **[建立 x64 （64位） DaRT 影像**]。

    -   如果您要建立32位電腦的復原影像，請選取 **[建立 x86 （32位） DaRT 影像**]。

2.  在 [**指定 windows 10 &lt; 64 位或32位 &gt; 安裝媒體的根路徑**] 方塊中，輸入 Windows 10 安裝檔的路徑。 使用與您要建立之復原影像之體系結構相符的路徑。

3.  按 **\[下一步\]**。

## 選取要納入恢復影像的工具


在 [工具] 頁面上，您可以選取要包含在復原影像上的多個工具。 當使用者引導進入 DaRT 影像時，這些工具就會提供給使用者使用。 不過，如果您在建立 DaRT 影像時啟用遠端連線，不論您選擇要在影像中包含哪些工具，都能在支援中心工作人員連線至最終使用者的電腦時，提供所有工具。

若要限制使用者存取這些工具，但仍保留透過遠端連線檢視器對工具的完整存取權，請不要在 [工具] 頁面上選取這些工具。 最終使用者將只能使用遠端連線，而且將能夠看到您從復原影像排除的任何工具，但無法存取。

**選取要納入恢復影像的工具**

1.  在 [**工具**] 頁面上，選取您要包含在影像中的每個工具旁的核取方塊。

2.  按 **\[下一步\]**。

## 選擇是否允許由技術支援人員進行遠端連線


在 [遠端連線] 頁面上，您可以選擇讓技術支援人員在最終使用者的電腦上遠端連線到並執行 DaRT 工具。 [遠端連線] 選項會在 [診斷與修復工具組] 視窗中顯示為可用的選項。 技術支援人員建立遠端連線之後，就可以從遠端位置，在最終使用者電腦上執行 DaRT 工具。

**若要選擇是否允許由技術支援人員進行遠端連線**

1.  在 [**遠端**連線] 頁面上，選取 [**允許遠端**連線] 核取方塊以允許遠端連線，或清除該核取方塊以防止遠端連線。

2.  如果您已清除 [**允許遠端**連線] 核取方塊，請按 **[下一步]**。 否則，請移至下一個步驟，繼續設定遠端連線。

3.  選取下列其中一項：

    -   讓 Windows 選擇開啟的埠號碼。

    -   指定埠號碼。 如果您選取此選項，請在選項下方的欄位中輸入1到65535之間的埠號碼。 建立遠端連線時，會用到這個埠號碼。 建議將埠號碼設為1024或更高，以將衝突可能性降至最低。

4.  （選擇性）在 [**遠端連線歡迎**訊息] 方塊中，建立最終使用者建立遠端連線時接收的自訂訊息。 訊息最多可以為2048個字元。

5.  按 **\[下一步\]**。

    如需遠端執行 DaRT 工具的詳細資訊，請參閱[如何使用 DaRT 復原影像來復原遠端電腦](how-to-recover-remote-computers-by-using-the-dart-recovery-image-dart-10.md)。

## 在恢復影像中新增驅動程式


在 [高級選項] 頁面的 [驅動程式] 索引標籤上，您可以新增修復電腦時可能需要的其他裝置驅動程式。 這些通常會包括 Windows 10 不提供的儲存空間或網路控制器。 在建立影像時會安裝驅動程式。

**重要** 當您選取要包含的驅動程式時，請注意，DaRT 不支援無線連線（例如藍牙或 802.11 a/b/n）。

 

**在恢復影像中新增驅動程式**

1.  在 [**高級選項**] 頁面上，按一下 [**驅動程式**] 索引標籤。

2.  按一下**新增**。

3.  流覽至要新增的驅動程式檔案，然後按一下 [**開啟**]。

    **記事** 驅動程式檔案是由儲存空間或網路控制器的製造商所提供。

     

4.  針對您要加入的每個驅動程式，重複步驟2和3。

5.  按 **\[下一步\]**。

## 將 WinPE 選用套件新增到復原影像


在 [高級選項] 頁面的 [WinPE] 索引標籤上，您可以將 WinPE 選用套件新增至 DaRT 映射。 這些套件是 Windows ADK 的一部分，這是 DaRT 復原映像嚮導的安裝先決條件。 您可以選取的工具全都是選用的。 系統會根據您在 [工具] 頁面上選取的工具，自動新增任何所需的套件。

您也可以指定暫存空間的大小。 [暫存空間] 是指為執行 DaRT 所設定的 RAM 磁碟空間量。 如果使用者無法使用使用者的硬碟，暫存空間就很有用。 如果您正在執行其他工具和驅動程式，您可能會想要增加暫存空間。

**將 WinPE 選用套件新增到復原影像**

1.  在 [**高級選項**] 頁面上，按一下 [ **WinPE** ] 索引標籤。

2.  選取您要包含在影像中的每個套件旁的核取方塊，或按一下 [**名稱**] 核取方塊以選取所有套件。

3.  在 [**暫存空間**] 欄位中，選取要為執行 DaRT 指派的 RAM 磁碟空間量（如果最終使用者的硬碟無法使用）。

4.  按 **\[下一步\]**。

## 新增故障分析程式的調試工具


如果您在 ISO 映像中包含 [崩潰分析程式] 工具，您也必須包含適用于 Windows 的調試工具。 在 [高級選項] 頁面的 [崩潰分析器] 索引標籤上，您可以輸入 Windows 10 調試工具的路徑，這些工具會讓 Analyzer 用來分析記憶體傾印檔案。 您可以使用執行 DaRT 復原映射嚮導之電腦上的工具，或者您可以使用終端使用者電腦上的工具。 如果您決定在最終使用者電腦上使用這些工具，請記住，您診斷的每個電腦都必須安裝調試工具。

如果您已安裝 Microsoft Windows 軟體發展工具組（SDK）或 Microsoft Windows 開發套件（WDK），則預設會將 Windows 10 調試工具新增到復原影像，且會自動填入調試工具的路徑。 如果檔案位於預設檔路徑所指示的位置以外的位置，您可以變更 Windows 10 調試工具的路徑。 嚮導中的連結可讓您下載並安裝 Windows 版調試工具（如果尚未安裝）。

若要下載 Windows 調試工具，請參閱[windows 版調試工具](https://go.microsoft.com/fwlink/?LinkId=266248)。 將調試工具安裝到預設位置。

**記事** DaRT 嚮導會檢查登錄機碼中的工具 `HKLM\Software\Microsoft\Windows Kits\Installed Roots\WindowsDebuggersRoot` 。 如果沒有登錄值，嚮導會根據您的系統架構來尋找下列其中一個位置：

`%ProgramFilesX86%\Windows Kits\10.0\Debuggers\x64`

`%ProgramFilesX86%\Windows Kits\10.0\Debuggers\x86`

 

**新增故障分析程式的調試工具**

1.  按一下 [**高級選項**] 頁面上的 [**崩潰分析**程式] 索引標籤。

2.  可選按一下 **[下載調試工具**]，下載適用于 Windows 的調試工具。

3.  選取下列其中一個選項：

    -   **包括 Windows 10 &lt; 64 位或32位 &gt; 調試工具**。 如果您選取此選項，請流覽並選取工具的位置（如果路徑尚未顯示）。

    -   **使用系統中正在調試的調試工具**。 如果您選取這個選項，在問題電腦上找不到 Windows 版調試工具時，系統會無法使用 [崩潰分析程式]。

4.  按 **\[下一步\]**。

## 選取要建立的復原圖像檔案類型


在 [建立影像] 頁面上，您可以為復原影像選擇輸出檔案夾，輸入影像名稱，然後選取要建立的 DaRT 復原影像檔案類型。 在恢復影像建立程式期間，會將 Windows 來源檔案解包、將 DaRT 檔案複製到該檔案，然後再將影像「重新封包」成您在此頁面上選取的檔案格式。

可用的圖像檔案類型有：

-   **Windows Imaging 檔案（WIM）** -用於將 DaRT 部署到開機前執行環境（PXE）或本機分區）。

-   **國際標準組織（ISO）** -用來部署到 CD 或 DVD，或用於虛擬電腦（VM） s）。 此嚮導要求 ISO 影像的副檔名為 .iso，因為燒錄 CD 或 DVD 的大多數程式都需要該副檔名。 如果您沒有指定其他位置，則會在您的桌上型電腦上使用名稱 DaRT10 建立 ISO 影像。

-   **PowerShell 腳本**–建立 dart 復原影像，其命令主要提供與您可以使用 DaRT 復原影像嚮導選取的選項。 此腳本也可讓您在 DaRT 恢復影像中新增或變更檔案。

如果您在此頁面上選取 [編輯影像] 核取方塊，您可以在影像建立過程中自訂復原影像。 例如，您可以變更「winpeshl.ini」檔案以建立自訂啟動順序或新增協力廠商工具。

**選取要建立的復原圖像檔案類型**

1.  在 [**建立影像**] 頁面上，按一下 **[流覽]** ，選擇影像檔的輸出檔案夾。

    **記事** 根據您所選取的工具和您在嚮導中新增的檔案而定，影像的大小會有所不同。

     

2.  在 [**影像名稱**] 方塊中，輸入 DaRT 復原影像的名稱，或接受預設名稱（DaRT10）。

    嚮導會以這個名稱在輸出路徑中建立子資料夾。

3.  選取您要建立的圖像檔案類型。

4.  選擇下列其中一項：

    -   若要在建立影像檔案之前變更復原影像中的檔案，請選取 [**編輯影像**] 核取方塊，然後按一下 [**準備**]。

    -   若要在不變更檔案的情況下建立恢復影像，請按一下 [**建立**]。

5.  

    按 **\[下一步\]**。

## 編輯復原影像檔


您可以在 [建立影像] 頁面上選取 [編輯影像] 核取方塊，以編輯復原影像。 在恢復影像準備好進行編輯之後，您就可以在建立可啟動的媒體前新增及修改復原映射檔。 例如，您可以建立啟動的自訂訂單，新增各種協力廠商工具等等。

**若要編輯復原映像檔**

1.  在 [**編輯影像**] 頁面上，按一下 [在 Windows 資源管理器中**開啟**]。

2.  在對話方塊中所列的資料夾中建立子資料夾。

3.  將您想要的檔案複製到新的子資料夾，或移除您不想要的檔案。

4.  按一下 [**建立**]，開始建立恢復影像。

## 產生復原映像檔


在 [產生檔案] 頁面上，系統會針對您在 [建立影像] 頁面上選取的檔案類型產生 DaRT 恢復影像。

**產生復原映像檔**

-   在 [**產生**檔案] 頁面上，按一下 **[下一步]** 來產生復原圖像檔案。

## 將恢復影像複製到 CD、DVD 或 USB


在 [建立可引導媒體] 頁面上，您可以選擇性地將圖像檔案複製到 CD、DVD 或 USB 快閃記憶體磁片磁碟機（UFD）。 您也可以重新開機嚮導，從這個頁面建立其他可引導媒體。

**記事** 此工具本身不支援開機前執行環境（PXE）和本機映射部署，因為它們需要其他企業工具，例如 System Center Configuration Manager 伺服器和 Microsoft 開發工具套件。

 

**若要將恢復影像複製到 CD、DVD 或 USB**

1.  在 [**建立可引導媒體**] 頁面上，選取您要複製的 iso 檔案。

2.  插入 CD、DVD 或 USB，然後選取磁片磁碟機。

    **記事** 如果無法辨識某個磁片磁碟機，且您安裝了新的磁片磁碟機，您可以按一下 [重新整理 **]，強迫**嚮導更新可用的磁片磁碟機清單。

     

3.  按一下 [**建立可引導媒體**] 按鈕。

4.  若要建立另一個復原影像，請按一下 [重新開機]，如果您已完成建立您想要的所有媒體，請按一下 [**關閉**]。
